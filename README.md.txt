# LensNinja

LensNinja is an AI-powered tool for finding the best-priced camera lenses online using web scraping, search filters, and image verification.

## Features

- Real-time search from eBay and MPB
- AI metadata tagging (brand, mount, price range)
- Image verification with OpenCV
- React + Tailwind frontend
- FastAPI backend API

## Project Structure

- `/backend` – FastAPI backend + scrapers + AI
- `/frontend` – React UI with filters and search
- `/data` – Temporary storage for scraped data
